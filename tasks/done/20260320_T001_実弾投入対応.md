# 実弾投入対応（OANDA BrokerAdapter）

## 背景・目的
FX Sim v1のペーパートレードをOANDA REST API v20経由で実弾取引可能にする。
BrokerAdapter抽象層で銘柄単位にペーパー/実弾を切替可能にする。
