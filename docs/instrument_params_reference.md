# instrument_params パラメーター参照ドキュメント

> 最終更新: 2026-03-29
> データソース: Cloudflare D1 `fx-sim-v1-db`（スキーマ v229）

## 概要

`instrument_params` テーブルは銘柄ごとのトレード戦略パラメーターを管理する。
AIによる自動レビュー（`PARAM_REVIEW_v1_GPT`）が定期的にパラメーターを調整する。

**管理銘柄数**: 46銘柄（FX4 + 円クロス3 + 株価指数6 + 暗号1 + コモディティ5 + 日本株19 + 米国株8）

> **注**: 2026-03-25以降に追加された銘柄（円クロス3 + 日本株19 + 米国株8 = 30銘柄）は
> `getDefaultParams()` による自動初期化値（v1）の状態。AIレビューはトレードデータ蓄積後に発動。

---

## AI調整の仕組み

`param-review.ts` のUPDATE文を確認した結果、**`review_trade_count` を除く全パラメーターがAI調整対象**（UPDATE SET句に含まれている）。

現在「全銘柄で共通値」になっているパラメーターは、**設計上は調整対象だがトレードデータ蓄積中のためまだ変更されていない**状態。

| 状態 | 説明 |
|---|---|
| 🔴 **調整済み** | AIが既に銘柄ごとに異なる値に変更した |
| 🟡 **調整対象・未変更** | UPDATE対象だが初期値のまま（データ蓄積中） |
| ⚫ **調整対象外（固定）** | UPDATE文に含まれておらず設計上変更されない |

## メモ

- `review_trade_count` を除く**全パラメーターがAI調整対象**（`param-review.ts` のUPDATE SET句で確認済み）
- 現在「共通値」なのは「設計上固定」ではなく「トレードデータ蓄積中でまだ変更されていない」状態
- 特に注目すべき未調整パラメーター:
  - `strategy_primary`（全銘柄 `mean_reversion` のまま → トレンド系銘柄でも未切替）
  - シグナル重み（`w_*`）→ 銘柄特性に応じた重み調整が今後起こりうる
  - `atr_sl_multiplier`（固定1.5）→ 調整されると実効RR比が変わる
- `atr_tp_multiplier / atr_sl_multiplier` の比 = 実際のRR比（例: 3.0/1.5 = 2.0）
- `min_rr_ratio`（1.5）はエントリーフィルターだが、理論RRは常に2.0以上なので現時点では実質機能していない
